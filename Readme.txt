Installation:
You can directly run the project and the game will automatically start

Instructions:

Home Page:
You will be given the Instructions in how to play the game and all you need to do is click the play game button and it should route you to the game.

The mechanics of the game is easy. Each player has a change to flip 2 cards and compare them. You lose the turn if you flip 2 cards that are not thesame and gain a point if you do.
Flip the cards up until all has been flipped and the player who has the highest points win.
